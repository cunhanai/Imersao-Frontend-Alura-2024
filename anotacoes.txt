PARA CRIAR UMA API FAKE:
Utilizaremos o json-server

Instalação:
    npm i json-server -g

Para inicializar:
    json-server --watch [caminho do arquivo] --port [porta escolhida]

- O caminho é o arquivo que será a API fake
- A porta pode ser a que você quiser

Exemplo de inicialização:
    json-server --watch api-artists\artists.json --port 3000
